Керівник роботи: Лавренюк І.В.

2. Тема: «Створення кулінарного сайту “В гостях у господарочки”».

Сайт є веб-застосунком для перегляду та додавання кулінарних рецептів із підтримкою коментарів, системи реєстрації та інтегрованим чат-ботом на основі штучного інтелекту.
​
Серверна частина реалізована на ASP.NET Core (C#), база даних — MySQL, мікросервіс чат-бота — на Django(Python) з використанням API Cohere.

Проєкт розгорнуто у хмарному середовищі Railway.

3. Процесор: Intel Core i5 або аналогічний (2 ядра і більше).
Оперативна пам’ять: не менше 4 ГБ.
Вільне місце на диску: не менше 10 ГБ.
Стабільне підключення до Інтернету.

4. Операційна система: Windows 10/11 (або сумісна ОС для запуску .NET 8, Python 3.10, MySQL).
​
Встановлені системні компоненти: .NET 8 Runtime / SDK.

5. .NET 8 SDK (або новіше).
Visual Studio 2022 або Visual Studio Code (для запуску з вихідних кодів).
Python 3.10+, pip.
MySQL Server 8.0+, MySQL Workbench (якщо використовувати локальну БД).
Будь-який сучасний браузер (Chrome, Edge, Firefox тощо).

6. Файли запуску
Основний сайт:

project.dll з інструкцією запуску через dotnet project.dll.
Або адреса Railway: https://kvalificationwork-production.up.railway.app/

7.  Необхідно налаштувати рядок підключення до бази даних у файлі appsettings.json (адреса сервера, порт, назва бази, користувач і пароль). Рекомендується не зберігати пароль у відкритому вигляді у публічному репозиторії.

Перед запуском мікросервісу необхідно встановити залежності з requirements.txt командою pip install -r requirements.txt.
​
Для коректної роботи чат-бота потрібен дійсний API-ключ Cohere, який задається через змінні середовища або конфігураційний файл.
